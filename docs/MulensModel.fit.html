MulensModel.fit module¶

class MulensModel.fit.Fit(data=None, magnification=None, n_sources=None)¶

Bases: object

fit_fluxes(): Performs a least squares linear fit

(np.linalg.lstsq) to the data for the flux parameters. i.e. given the data y and magnifcation A, solves for f_source and f_blend:

y = f_source * A + f_blend

Allows for zero blending.

get_input_format(data): Returns the model magnifications in the
same flux system as data.
