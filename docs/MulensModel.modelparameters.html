MulensModel.modelparameters module¶

class MulensModel.modelparameters.ModelParameters(t_0=None, u_0=None, t_E=None, rho=None, s=None, q=None, alpha=None, pi_E=None, pi_E_N=None, pi_E_E=None, pi_E_ref=None)¶

Bases: object

A class for the basic microlensing model parameters (t_0, u_0, t_E, rho, s, q, alpha, pi_E). Can handle point lens or binary lens. pi_E assumes NE coordinates (Parallel, Perpendicular coordinates are not supported).

n_lenses¶

number of objects in the lens system

t_0¶

The time of minimum projected separation between the source and the lens center of mass.

u_0¶

The minimum projected separation between the source and the lens center of mass.

t_E¶

The Einstein timescale. An astropy.Quantity. “day” is the default unit.

rho¶

source size as a fraction of the Einstein radius

alpha¶

The angle of the source trajectory relative to the binary lens axis (or primary-secondary axis). Measured CW/CCW (TBD). An astropy.Quantity. “deg” is the default unit. TBD - make sure CW/CCW convention is according to Skowron+11 appendix A

q¶

mass ratio of two lens components

s¶

separation of two lens components relative to Einstein ring size

pi_E¶

The microlens parallax vector. May be specified either relative to the sky (“NorthEast”) or relative to the binary lens axis (“ParPerp”). “NorthEast” is default.

pi_E_N¶

The North component of the microlens parallax vector.

pi_E_E¶

The East component of the microlens parallax vector.
