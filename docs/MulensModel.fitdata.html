MulensModel.fitdata module¶

class MulensModel.fitdata.FitData(model=None, dataset=None, fix_blend_flux=False, fix_source_flux=False, fix_q_flux=False)¶

Bases: object

Performs a least squares linear fit for given dataset and model to determine the source flux(es) and (optionally) blend flux.

Arguments :
model: Model object
The model to fit to the data.
dataset: MulensData object
A single photometric dataset to be fitted.
fix_blend_flux: False or float
Default is False, i.e. allow the blend flux to be a free parameter. If set to a float, it will fix the blend value to that value.
fix_source_flux: False, float, or list
Default is False, i.e. allow the source flux to be a free parameter. If set to a float, it will fix the source value to that value. For binary source models, a list should be used to set the fluxes of the individual sources or fix one and not the other, e.g. [2.3, False] would fix f_source_0 to 2.3 but allow a free fit to f_source_1. (Not Implemented)
fix_q_flux: False or float
For binary source models, q_flux is the flux ratio between two components, i.e. q_flux = f_source_1 / f_source_0 Default is False, i.e. allow the source flux to be a free parameter. If set to a float, it will fix the source value to that value. (Not Implemented)
fit_fluxes()¶

Execute the linear least squares fit to determine the fitted fluxes. Sets the values of source_fluxes, blend_flux, and (if applicable):py:obj:~source_flux.

source_flux¶
Returns:float the fitted source flux. Only defined for models with a single source. See also source_fluxes

If None, you need to run fit_fluxes() to execute the linear fit.
source_fluxes¶
Returns:np.array the fitted source flux(es).

If None, you need to run fit_fluxes() to execute the linear fit.
blend_flux¶
Returns:the fitted blend flux or the value set by fix_blend_flux (see Keywords).

If None, you need to run fit_fluxes() to execute the linear fit.
q_flux¶

q_flux = f_source_1 / f_source_0

Returns:the ratio of the fitted source fluxes or the value set by fix_q_flux (see Keywords).

If None, you need to run fit_fluxes() to execute the linear fit.
