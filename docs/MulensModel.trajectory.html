MulensModel.trajectory module¶

class MulensModel.trajectory.Trajectory(times, parameters=None, parallax=None, coords=None, satellite_skycoord=None, earth_coords=None)¶

Bases: object

The (dimensionless) X, Y trajectory of the source in the source plane.

t_0_par is the reference time for the parallax vector. If not set, defaults to t_0.

For binary lens, the origin of the coordinate system is at the center of mass with higher mass at negative X and Y=0.

get_xy()¶

For a given set of parameters (a ModelParameters object), calculate the xy position of the source.
